# Repository Levels

This skill supports different levels of content repos.

## Level 1 — personal content repo
Example:
- personal portfolio
- artist archive
- solo practice repo

Recommended first-wave objects:
- person
- work
- node
- asset
- relation

## Level 2 — collective content repo
Example:
- team site
- collective works repo
- shared activity repo

Recommended first-wave objects:
- collective
- work
- node
- person
- relation

## Level 3 — distributed ecosystem repo
Example:
- shared content protocol repo
- multi-site source layer

Recommended objects:
- person
- collective
- work
- node
- asset
- writing
- venue
- spec
- relation

## Rule
Do not force every repo to hold every object type in its first wave.
Use the smallest stable level first.
