# Coder Upload Instructions

## What to upload

1. create a new repository named `VIRTURA-Content`
2. upload the contents of `VIRTURA-Content/` from this package to that repo
3. in `VIRTURA-SpacePort`, add a new station folder:
   `stations/content-protocol-station/`
4. upload the provided `README.md` into that station folder
5. do not merge schemas into existing frontend repos yet
6. keep the first version lightweight and documentation-first
