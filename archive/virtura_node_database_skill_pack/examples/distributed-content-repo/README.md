# Distributed Content Repo Example

This example shows how a shared protocol repo should use the skill.

## First-wave purpose
Not to replace public sites.
Instead, provide shared source objects for:
- portfolio
- collective
- spaceport
- newsroom
- viewer

## Good first-wave objects
- people
- collective
- works
- nodes
- relations

## Add later
- assets
- writings
- venues
- specs
