python3 -m http.server 8080

📖 安装说明

🐍

Python依赖

cd virtura_content_system
pip install -r cli/requirements.txt
pip install -r scripts/requirements.txt
📦

Node.js依赖

cd virtura_content_system/mcp_server
npm install

🎯 快速启动

# 1. 配置服务
cp config/mcp_config.example.yaml config/mcp_config.yaml

# 2. 启动MCP服务（默认作为stdio服务器）
cd virtura_content_system
./start_mcp_server.sh

# 或通过npm直接启动
cd mcp_server
npm start

🌐 配置Claude Desktop

// 在Claude Desktop配置文件中添加
{
  "mcpServers": {
    "virtura-content": {
      "command": "node",
      "args": ["/path/to/virtura_content_system/mcp_server/src/index.js"]
    }
  }
}

❓ 常见问题

为什么下载的是源代码而不是二进制文件？

原因：Virtura Node Weaver MCP是一个开源项目，主要使用Python和JavaScript开发，不需要编译为二进制文件。

解决方案：直接下载源代码，按照安装说明进行配置即可使用。

安装过程中遇到依赖问题怎么办？

原因：可能缺少某些系统依赖库。

解决方案：确保已安装最新版本的Python和Node.js，或尝试使用虚拟环境。

服务无法启动是什么原因？

原因：可能是端口被占用、配置文件错误或权限问题。

解决方案：检查端口是否被占用，验证配置文件格式，确保有足够的权限。

如何升级到最新版本？

方法1：使用git命令升级 

git pull origin main

方法2：下载最新版本并覆盖安装
