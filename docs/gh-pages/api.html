GET content_system_search

搜索内容系统中的对象

参数

参数 类型 必填 描述
query string 否 搜索关键词，匹配标题、描述、标签等
type string 否 按内容类型过滤
tags string[] 否 按标签过滤
year integer 否 按年份过滤
limit integer 否 返回结果数量限制，默认 50
offset integer 否 分页偏移量，默认 0

返回值

{
  "results": [
    {
      "id": "work:drop-flow",
      "type": "work",
      "title": "Drop Flow",
      "description": "交互式数字艺术作品",
      "tags": ["数字艺术", "交互"],
      "year": 2024
    }
  ],
  "total": 42,
  "offset": 0,
  "limit": 50
}

使用示例

// 搜索所有对象
content_system_search({})

// 搜索包含 "art" 的作品
content_system_search({
  "query": "art",
  "type": "work",
  "limit": 10
})

// 搜索 2024 年的展览节点
content_system_search({
  "type": "node",
  "year": 2024,
  "tags": ["展览"]
})

GET content_system_get

通过Typed ID获取单个对象的详细信息

参数

参数 类型 必填 描述
id string 是 Typed ID（格式：type:id）
include_relations boolean 否 是否包含关联关系，默认 true

返回值

{
  "id": "work:drop-flow",
  "type": "work",
  "title": "Drop Flow",
  "description": "交互式数字艺术作品",
  "summary": "探索数据可视化与用户交互的艺术实践",
  "tags": ["数字艺术", "交互", "数据可视化"],
  "year": 2024,
  "status": "published",
  "relations": [
    {
      "id": "relation:drop-flow-author",
      "type": "author_of",
      "target": "person:ewan-qian",
      "target_title": "Ewan Qian",
      "description": "主要创作者"
    },
    {
      "id": "relation:drop-flow-featured",
      "type": "featured_in",
      "target": "node:hangzhou-opening",
      "target_title": "杭州开幕展",
      "description": "首次展出"
    }
  ]
}

使用示例

// 获取对象及其关联
content_system_get({
  "id": "work:drop-flow"
})

// 仅获取对象不包含关联
content_system_get({
  "id": "person:ewan-qian",
  "include_relations": false
})

POST content_system_create

创建新的内容对象

参数

参数 类型 必填 描述
type string 是 内容类型
fields object 是 对象字段

fields 对象

字段 类型 描述
id string 可选自定义 ID
title string 对象标题
name string 对象名称（人物/集体）
description string 描述
summary string 简短摘要
tags string[] 标签
date string 日期（ISO 或 YYYY）
status string 状态

返回值

{
  "id": "node:test-node-abc123",
  "object": {
    "id": "node:test-node-abc123",
    "type": "node",
    "title": "测试节点",
    "tags": ["测试"],
    "created_at": "2024-04-13T10:30:00Z"
  }
}

使用示例

// 创建一个新作品
content_system_create({
  "type": "work",
  "fields": {
    "title": "New Art Piece",
    "description": "A description of the work",
    "tags": ["art", "new"],
    "date": "2026",
    "status": "published"
  }
})

// 创建一个新人物
content_system_create({
  "type": "person",
  "fields": {
    "name": "Jane Artist",
    "summary": "Multimedia artist based in NYC",
    "tags": ["artist", "multimedia"]
  }
})

POST content_system_link

在两个对象之间建立双向关联关系

参数

参数 类型 必填 描述
source_id string 是 源对象 Typed ID
target_id string 是 目标对象 Typed ID
relation_type string 否 关系类型，默认 "related"
fields object 否 额外关系字段

关系类型

创作关系

  • author_of - 作者
  • contributor_to - 贡献者

展示关系

  • featured_in - 特色展示
  • part_of - 属于

其他

  • references - 引用
  • related - 相关（默认）

返回值

{
  "relation_id": "relation:source-target-abc123",
  "relation": {
    "id": "relation:source-target-abc123",
    "source": "person:ewan-qian",
    "target": "work:drop-flow",
    "type": "author_of",
    "description": "Primary creator",
    "created_at": "2024-04-13T10:30:00Z"
  }
}

使用示例

// 建立作者关系
content_system_link({
  "source_id": "person:ewan-qian",
  "target_id": "work:drop-flow",
  "relation_type": "author_of",
  "fields": {
    "description": "Primary creator",
    "start_date": "2023"
  }
})

// 建立从属关系
content_system_link({
  "source_id": "work:timer",
  "target_id": "node:hangzhou-opening",
  "relation_type": "featured_in"
})

GET content_system_export

导出内容为指定格式

参数

参数 类型 必填 描述
query string 否 搜索关键词
type string 否 内容类型过滤
tags string[] 否 标签过滤
ids string[] 否 指定 ID 列表导出（忽略其他过滤器）
format string 否 格式：json, csv, markdown（默认 json）
limit integer 否 导出数量限制，默认 100

返回值

{
  "content": "[导出的内容字符串]",
  "format": "json",
  "count": 42
}

使用示例

// 导出所有作品为 JSON
content_system_export({
  "type": "work",
  "format": "json"
})

// 导出特定对象为 Markdown
content_system_export({
  "ids": ["work:drop-flow", "work:timer", "node:hangzhou-opening"],
  "format": "markdown"
})

// 导出 2024 年的展览为 CSV
content_system_export({
  "type": "node",
  "tags": ["展览"],
  "year": 2024,
  "format": "csv"
})

错误处理

常见错误码和处理方法

错误类型 描述 处理方法
Invalid ID ID格式不正确或对象不存在 检查ID格式是否正确（type:id），确认对象存在
Validation Error 字段验证失败 检查必填字段是否都已提供，字段类型是否正确
Permission Denied 无权限操作 检查安全配置，确认是否有操作权限
Already Exists 对象已存在 使用不同的ID，或不提供ID让系统自动生成
