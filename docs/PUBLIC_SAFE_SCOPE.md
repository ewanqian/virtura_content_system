# Public-Safe Scope

This repository is a public-safe release layer derived from a larger local working system.

## Included

- public project description
- public-safe framing of Node Weaver
- lightweight sample data
- minimal example export script

## Excluded

- local absolute paths
- internal workspace contracts
- agent supervision and evidence ledgers
- private review notes
- unpublished or uncertain asset bundles
- large local viewer payloads
- internal archive backups

## Release Rule

This directory should only contain material that is safe to publish as a standalone public repository.

If a file contains:

- local machine paths
- internal process notes
- handoff instructions
- unpublished asset locations
- operational or personal-only metadata

it should stay in the internal working repository, not here.
