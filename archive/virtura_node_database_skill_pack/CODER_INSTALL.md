# Coder Install Instructions

## Put this skill into your repo
Recommended location:

```text
skills/node-database-system/
```

## Minimum files to keep
- SKILL.md
- QUICKSTART.md
- AGENT_WINDOW_PROTOCOL.md
- OUTPUTS.md
- templates/
- REPOSITORY_LEVELS.md

## First run recommendation
1. read `REPOSITORY_LEVELS.md`
2. decide which repo level you are in
3. produce `OBJECT_INVENTORY.csv`
4. produce `STARTER_DATABASE.json`
5. produce `NOTE_PLAN.md`
6. validate object / view distinction

## Do not do first
- do not rewrite the whole frontend
- do not build the viewer first
- do not generate too many object types at once
