🚀 MCP 核心工具

5个核心API，覆盖内容管理全流程

🔍 content_system_search

强大的搜索功能，支持关键词、类型、标签、年份等多维度过滤，返回标准化结果。

📄 content_system_get

通过Typed ID获取单个对象的详细信息，包含所有关联关系，支持递归查询。

➕ content_system_create

创建新内容对象，支持所有类型，自动生成全局唯一ID，完整字段验证。

🔗 content_system_link

在对象间建立双向关联关系，支持多种关系类型，可附加描述和元数据。

📥 content_system_export

导出内容为JSON、CSV、Markdown格式，支持灵活的过滤条件和批量操作。

📋 content_system_list_types

获取系统支持的所有内容类型列表，包含类型定义和属性信息。
